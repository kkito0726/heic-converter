# .heicファイルの変換CLI

## 要望

- .heicファイルはそのままではApple以外のデバイスでは画像を閲覧できないので簡単に.jpgや.pngに変換できるCLIツールを開発する。
- 基本は対話形式で操作を進められるようにして変換できる拡張子は幅広く対応できるようにする。チェックボックスを埋めるようにして変換したい拡張子を複数選択できる
- .heicファイルのパスが渡されたら単一ファイルの変換し、フォルダのパスが渡されたらフォルダ内のheicファイルを全て変換する。
- リッチなCLIのように見た目もこだわる、配色や最初にアスキーアートを入れるなど

## アーキテクチャ

- Go 1.26.4
- devcontainerで開発環境を整える
- Goの単一ビルドで動作することを第一優先 (ユーザーの環境に依存しない)
- クリーンアーキテクチャを採用　domain層にinfra層のインターフェースを定義してinfra層がそれを実装する. (別ライブラリを使用したくなった時に簡単に実装を入れ替えられる)
- 入力層もinterfaceで抽象化してAPIとしても起動できるように将来的にしたい (webでもCLIでも対応できる設計)
